Physics Timer by daball

A physics timer that works in Windows 2000 SP3/XP SP2 and Vista/7 out-of-the-box.

About the Project

I started the project when I needed a good timer for the Physics course that would work in Windows 7 64-bit. So, I decided to implement a solution.

This program was written and the project is maintained at Github by David Ball. See web site at http://www.daball.me/. The project's home at Github is here: https://github.com/daball/Physics-Timer.

The original concept was by David Smith, Physics instructor at VHCC. This is an adaptation of the original concept for the .Net Framework 2.0.

User Guide

  1. Download the zip or tar archive file for the binary release.
  2. Extract/expand the archive file into a folder.
  3. Run the Timer.Net.exe file from the new folder.
  4. Use the Start Timer button to start the timer. The button becomes Stop Timer when you press it, and when clicked it will pause the timer.
  5. Once the timer is running, use the Record Lap Interval button to store your intervals into the data sheet.
  6. At the bottom of the screen, there are two more buttons. The Copy .CSV to Clipboard (Report) button generates a comma-seperated values worksheet to your Windows clipboard. The Save .CSV File (Report) button generates that same worksheet to a file that you specify.
  7. The Reset button will erase everything on the sheet and reset the timer to zero.
